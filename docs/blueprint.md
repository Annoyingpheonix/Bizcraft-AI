# **App Name**: BizCraft AI

## Core Features:

- Ad Generation: Generate marketing ad copy and visuals for various platforms, utilizing AI to tailor content based on business type and target audience. The LLM is capable of using a tool to incorporate recent high-performing marketing headlines into the prompt, for improved ad generation performance.
- Infographic Design: Create visually appealing infographics from provided data and text, with AI assistance for layout suggestions and content summarization.
- Logo Creation: Generate unique logo designs based on user input, including business name, industry, and desired aesthetic. Use a generative model that is trained to reject copyrighted or trademarked images.
- Chart and Graph Generation: Automatically generate charts and graphs from uploaded data, offering various customization options for visual representation.
- Whitepaper Outline Tool: Generate a structural outline and sections based on topic using LLM. Then suggest keywords that may be present, for SEO enhancement.
- Resource Export: Download created resources in various formats (PNG, JPG, PDF, SVG) for easy sharing and use.

## Style Guidelines:

- Primary color: Navy blue (#2E3192) to evoke trust and professionalism.
- Background color: Light gray (#F0F2F5) for a clean and modern look.
- Accent color: Teal (#00A3AD) to add a touch of creativity and innovation.
- Headline font: 'Poppins' (sans-serif) for a modern and readable heading.
- Body font: 'PT Sans' (sans-serif) for clear and accessible text.
- Use minimalist icons representing various business concepts.
- Employ a clean, grid-based layout to ensure a professional and organized presentation of information.
